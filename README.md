# Baigiamasis-front-end

Local Setup 
Requirements:
Visual Studio Code;
node.js 18+;
MongoSV atlas;
(make sure the back-end is running on http://localhost:3000);
Libary:
react: 19.1.1,
jwt-decode: 4.0.0;

Project Setup:
git clone https://github.com/erikaspipas99/baigiamasis.git
cd baigiamasis back-end
npm install
npm run dev
